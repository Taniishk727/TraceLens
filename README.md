# TraceLens

## Interactive OSINT Investigation Platform

TraceLens is an investigation-centric OSINT platform that aggregates publicly available information from multiple sources into a unified Investigation Canvas.

## Features

- Entity-based investigations
- Interactive Investigation Canvas
- Case Management
- Evidence Tracking
- Report Generation

## Tech Stack

- Flask
- SQLite
- HTML
- Bootstrap
- JavaScript
- Cytoscape.js
